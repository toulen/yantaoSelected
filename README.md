# yantaoSelected
Jquery下拉菜单插件，模拟下拉菜单
在HTML中引入js和CSS
之后编写HTML：如：
<div id="yantao-selected">
    <div class="select-title-info">
        <span class="text">请选择</span>
        <span class="icon f_right">▼</span>
    </div>

    <div class="select-options" style="display: none;">
        <ul>
            <li data-val="1">苹果</li>
            <li data-val="2">梨子</li>
        </ul>
    </div>
</div>
